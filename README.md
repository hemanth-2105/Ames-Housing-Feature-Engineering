# Ames Housing Price Prediction

## Project Overview
This project predicts house prices using the Ames Housing dataset. The project includes data exploration, feature engineering, preprocessing, and a Linear Regression model built with Scikit-learn.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

## Machine Learning Algorithm
- Linear Regression

## Project Workflow
1. Dataset Understanding
2. Data Cleaning
3. Ordinal Encoding
4. Feature Engineering
5. Data Validation
6. Data Preparation
7. Model Training
8. Model Evaluation

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Files
- `ames_housing.ipynb` – Jupyter Notebook
- `AmesHousing.csv` – Dataset
- `House_Price_Model.pkl` – Saved trained model (optional)
